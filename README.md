# Experimental Physics Data Analysis (Python)

Este repositório reúne scripts desenvolvidos em **Python** para análise de dados experimentais e geração de gráficos em experimentos de **Física Experimental**, com foco no estudo de **fenômenos eletromagnéticos**.

Os códigos foram utilizados para processar dados coletados em laboratório, realizar cálculos numéricos e gerar gráficos comparando resultados experimentais com modelos teóricos.

## Tecnologias utilizadas

- Python
- NumPy
- Matplotlib

## Estrutura do repositório

Cada pasta corresponde a um experimento realizado durante a disciplina. Dentro de cada diretório estão os scripts responsáveis pelo processamento dos dados e geração dos gráficos utilizados nas análasises.
